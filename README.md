# Phishing URL Detector

## Overview

A simple Python project that identifies potentially malicious phishing URLs using basic heuristic rules.

---

## Features

- Detect IP-based URLs
- Detect @ symbol
- Detect long URLs
- Detect suspicious keywords

---

## Technologies

- Python
- Regular Expressions

---

## Detection Rules

- IP Address in URL
- URL Length
- @ Symbol
- Suspicious Words

---

## Future Improvements

- Machine Learning
- URL Reputation API
- Browser Extension

---

## Learning Outcome

Understand how phishing URLs are identified and learn the basics of secure web browsing.
